Welcome to gene.iobio

An app for interrogating disease-causing variants

  • Type in the gene name BRCA2 and select from the drop down list.
    Click on next arrow > to continue to next step.
  • This shows the gene model for the canonical transcript.
  • Select 'Single' to view variants for a single sample. Select 'Trio' when performing trio analysis with variants from proband, mother, and father.
    In this tour, we will be analyzing the variants from a single sample, so press the 'Single' button.
  • This is where the 'Proband' variant file is selected. Select 'Platinum Trio' under 'Sample datasets' and variant file for NA12878 from the Illumina Platinum data set will be selected.
    When loading your own dataset, select 'Enter URL for VCF' if the data is hosted from a website. You need to specify the URL for the gzipped vcf (.vcf.gz). The tabix file (.vcf.gz.tbi) must exist in the same web directory.
    Select 'Choose VCF File' if your variant file resides on your file system. You must select both the gzipped vcf (.vcf.gz) and the tabix (.vcf.gz.tbi) files at the same time.
  • This is where the 'Proband' alignment file is optionally selected. Select Select 'Platinum Trio' under 'Sample datasets' and the alignment file for NA12878 will be selected.
  • Click 'Load' to load all of the selected data into the app.
  • Click the minimize button to hide the data selection card and increase the space available for analysis.
  • Click the maximize button to show the full view for the data selection card.
  • The variants for the gene are annotated using SnpEff and VEP to determine functional impact and effect, the allele frequencies from 1000 Genomes and ExAC are retrieved, and queries to ClinVar flag any variants with clinical significance. All of this processing occurs in real-time. Hover over the variants to see more information. Click on a variant to lock, allowing links to be clicked in the tooltip.
  • When an alignment file is specified, the base coverage chart is displayed, allowing the user to identify regions of abnormal or unexpected sequence coverage.
  • Push the 'Call from alignments' button to call variants using Freebayes. Variants not present in the VCF file will be displayed above the loaded variants.
  • The variants can be filtered by mousing over a specific label. To apply multiple filters, click on multiple labels to toggle on or off.
  • The variants are ranked according to different criteria. These criteria can be reordered by clicking on their respective arrows. Click on a column to see the variant in the variant chart below. Unclick the column to toggle back to all variants.
  • Other transcripts for this gene can be selected from this dropdown.
  • Drag and drop to select a region to zoom into. To zoom back out, click anywhere on the gene model.
  • Adjust this value to display modify the size of displayed DNA upstream and downstream of the gene.
  • You can run the tour any time by selecting 'Tour' from the 'Learn more' dropdown.
    This is the end of the tour. Click the close button to end or click on next arrow > to re-run the tour.
