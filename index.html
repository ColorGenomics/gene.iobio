Welcome to gene.iobio

An app for discovering disease-causing variants

  • Type in the gene name BRCA2.
  • This shows gene model for the canonical transcript.
  • Select 'Single' when to view one sample's variants. Select 'Trio' to analyze affect child, unaffected mother, and unaffected father.
  • This is where you specify the variant file for the Proband. Select 'Load from URL' and the variant file for Platinum data set NA12878 will load.
  • This is where you specify the alignment (bam) file for the Proband (optional). Select 'Load from URL' and the alignment file for the Platinum data set NA12878 will load.
  • The variants for the gene are annotated using snpEff to determine functional impact and effect, the allele frequencies from 1000 Genomes and ExAC are retrieved, and queries to ClinVar flag any variants with clinical signifigance. All of this processing occurs in real-time.
  • When an alignment file is specified, the base coverage chart is displayed, allowing the user to spot regions of insufficient sequence coverage.
  • Push the Find Missing Variants link to call variants using the Freebayes caller. New variants discovered with this call will be displayed above the loaded variants.
  • The variants can be filtered by mousing over a specific label. To apply multiple filters, click on mutiple labels to toggle on or off.
  • The variants are ranked according to different criteria. The criteria can be rearranged by clicking on the arrow. Click on the column to see the variant in the variant chart (next card down)
  • You can select another transcript from this dropdown.
  • Drag and drop to select a region to zoom into. To zoom back out, click anywhere on the gene model.
  • Adjust this value to display more or less of the upstream and downstream region of the gene.
